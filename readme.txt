=== Description ===
Этот плагин создает базу данных для хранения информации о фильмах и сериалах. А еще плагин умеет импортировать данные с kinopoisk и отображать видеоплеер на отдельной странице. Для этого достаточно указать ID фильма, плагин внесет его в базу и импортирует плеер для него. Плагин адаптирован под шаблон Twenty Seventeen, настройки отображения находится в файле core.php

=== Installation == 
Скопируйте папку плагина в директорию wp-content/plugins и активируйте его в админ-панели.
